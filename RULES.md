# RULES.md — スライド作成の憲法
## ジム・クレイマーのマッドマネー要約チャンネル専用ルール

> このファイルはすべてのスライド作成に必ず適用する。内容の変更は禁止。
> plan.md と組み合わせて使用すること。

---

## 1. 基本仕様

| 項目 | 規定値 |
|------|--------|
| スライド比率 | 16:9（幅 25.4cm × 高さ 14.29cm） |
| python-pptx実装 | 幅10インチ × 高さ5.63インチ |
| スライド枚数 | **最大12枚**（厳守） |
| 1枚目 | 必ず「本日の主要トピック整理」にする。タイトルサムネイルは作らない |
| 時間表示 | スライド本体には記載しない（plan.mdの管理用メモのみ） |
| **出力先** | **すべてのPPTXファイルは`outputs`フォルダに保存すること**（厳守） |

### スライド構造の標準化
- **ヘッダーバー**: 高さ0.9インチ、ダークネイビー背景、白文字タイトル（24pt Bold）
- **利用可能なコンテンツエリア**: 5.63" - 0.9" = **4.73インチ**（この範囲内に全要素を収める）
- **背景グレーボックス**: `#F5F7FA`でカラム・カードの背景を統一

|------|------|-------------|-------|---------|

- **ポジティブ（緑）とネガティブ（赤）は意味を持って使う**。装飾目的での使用禁止
- 「買い推奨」「強み」「成長」→ `#006B3F`
- 「売り推奨」「リスク」「マイナス数値」→ `#C41E3A`
- グラデーション・装飾的な色使いは禁止。フラットデザインを維持する

## 3. フォント設定

### フォントファミリー
```
日本語: 'Noto Sans JP', 'Meiryo', 'Hiragino Sans', sans-serif
英数字: 'Calibri', 'Arial', sans-serif
lang属性: ja（PDF出力時の漢字文字化け防止）
```

### フォントサイズ規定

| 要素 | サイズ | ウェイト | 備考 |
|------|--------|---------|------|
| ヘッダータイトル | 24pt | Bold | ネイビー背景のヘッダーバー内 |
| スライドタイトル（独立） | 36〜40pt | Bold | ヘッダーなしレイアウトの場合 |
| サブタイトル | 20〜22pt | Regular | タイトル直下 |
| カラム・カード見出し | 13〜16pt | Bold | セクション内の小見出し |
| 本文・箇条書き | 11〜15pt | Regular | 内容量に応じて調整 |
| ライトニング・ラウンド銘柄名 | 13pt | Bold | タイトル行 |
| ライトニング・ラウンド詳細 | 11pt | Regular | 説明行 |
| 強調数字（大） | 60pt | Bold | 大数字レイアウト使用時 |
| 強調数字（中） | 36〜48pt | Bold | カード内の数字 |
| 注釈・補足 | 10〜12pt | Regular | スライド下部の小テキスト |
| クレイマーコメント | 12〜13pt | Regular/Italic | ネイビー背景ボックス内 |

### フォント使用ルール
- **本文テキストは左揃え**（タイトルのみ中央揃えを許可）
- 箇条書きは字下げ1段まで（多段ネストは読みにくいため禁止）
- 太字は見出しと強調箇所のみ。本文中の多用禁止

## 4. レイアウト規定

### 共通マージン・スペーシング（python-pptx実装値）
- スライド左右の余白：**0.4〜0.6インチ**（約1.0〜1.5cm）
- ヘッダーバー高さ：**0.9インチ**（固定）
- コンテンツ開始位置：**1.1インチ**（ヘッダー下0.2インチから）
- 利用可能な縦スペース：**最大4.3〜4.5インチ**（はみ出し厳禁）
- カラム間・カード間のギャップ：**0.1〜0.15インチ**（約0.25〜0.4cm）
- テキストボックス内マージン：**0.12インチ以上**
- 行間調整：`space_after`で2〜8pt（内容の密度に応じて調整）
- **重要**: すべての要素が縦5.63インチ以内に収まること（はみ出しチェック必須）

## 5. 視覚的ルール

### やること
- 各スライドに必ず**1つ以上の視覚的アクセント**を入れる（大数字・カラーボックス・罫線など）
- 強調したい数値は**必ず大きく表示**する（60pt Bold）
- ポジティブ／ネガティブの情報は**必ずカラーで区別**する（🟢緑=買い、🔴赤=売り）
- 「クレイマーの評価」などの引用はボックス囲み＋ネイビー背景＋白テキストで統一
  - **実装値**: `(0.4〜0.6, 4.6〜4.9, 8.8〜9.2, 0.6〜0.7)`、12〜13pt、margin_top=0.1", margin_left=0.3"
- すべてのカラム・カードには背景色`#F5F7FA`と罫線`#E0E0E0`を適用

### やってはいけないこと
- グラデーション・シャドウ・装飾的エフェクトの使用
- 1スライドに5種類以上のフォントサイズを混在
- テキストの右揃え（左揃えまたは中央揃えのみ）
- 画像・写真の挿入（絵文字・記号のみ許可）
- 1スライドに情報を詰め込みすぎる（迷ったらスライドを分割）
- **要素のはみ出し**（最重要）：すべての要素が縦5.63"以内に収まることを必ず確認

## 6. コンテンツルール

- **「買い推奨」「売り推奨」は必ずカラーで明示**（緑=買い、赤=売り）
- クレイマーの発言を引用する場合は「」で囲みボックス表示
- 数値データは出典・文脈が分かるよう必ずラベルを添える
- 投資情報である旨の注記を最終スライドに必ず入れる：
  ```
  ※本スライドは情報提供のみを目的としており、投資の勧誘を意図するものではありません。
  ```

## 7. PDF出力時の注意（日本語文字化け防止）

```css
font-family: 'Noto Sans JP', 'Meiryo', 'Hiragino Sans', sans-serif;
lang="ja"
```

- 必ずHTMLおよびCSS全体に`lang="ja"`を指定する
- 英数字もNoto Sans JPでカバーされるが、Calibriを優先指定すること
- LibreOfficeで変換する場合、フォントの埋め込みを有効にする

## 8. python-pptx実装時の注意事項

### 必須の関数・設定
```python
# スライドサイズ設定
prs.slide_width = Inches(10)
prs.slide_height = Inches(5.63)

# カラー定義
NAVY = RGBColor(26, 43, 74)
DEEP_GREEN = RGBColor(0, 107, 63)
ALERT_RED = RGBColor(196, 30, 58)
WHITE = RGBColor(255, 255, 255)
LIGHT_GRAY = RGBColor(224, 224, 224)
BG_GRAY = RGBColor(245, 247, 250)

# ヘッダー付きスライド作成関数
def add_slide_with_header(prs, title_text):
    slide = prs.slides.add_slide(prs.slide_layouts[6])
    header_shape = slide.shapes.add_shape(1, Inches(0), Inches(0), Inches(10), Inches(0.9))
    header_shape.fill.solid()
    header_shape.fill.fore_color.rgb = NAVY
    # タイトル設定（24pt Bold White）
    return slide
```

### はみ出し防止の計算式
- **利用可能な縦スペース** = 5.63" - 0.9" (ヘッダー) = **4.73"**
- コンテンツ開始位置：1.1"（ヘッダー下0.2"）
- コンテンツ終了位置：5.4"以内（余裕を持つ）
- カラム・カードの高さ = 利用可能スペース - マージン - 他要素の高さ

### word_wrap設定
- すべてのtext_frameに`word_wrap = True`を設定
- 長い文章が改行されるようにする

## 10. トラブルシューティング

### はみ出しが発生した場合の対処法
1. **フォントサイズを1〜2pt小さくする**（13pt → 11pt、15pt → 13ptなど）
2. **行間を詰める**（`space_after`を8pt → 6pt → 2ptへ調整）
3. **カラム・カードの高さを減らす**（4.3" → 4.0" → 3.3"など）
4. **項目数を減らす**（スライドを分割する）
5. **補足ボックスの位置を上げる**（4.7" → 4.5" → 4.0"など）

### ファイルが開かれていて保存できない場合
- 出力ファイル名を変更する（例：`_fixed.pptx`を付ける）
- PowerPointでファイルを閉じてから再実行
